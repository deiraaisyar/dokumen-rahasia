# OECD NEA Coding Competition

Welcome to the OECD NEA Coding Competition. This repository has all the files needed to support your submission please read it carefully. 

# Step 1. Nationality Compliance and Work Attestation

In this repository you will find a form labelled "Attestation.docx". BEFORE BEGINNING WORK please complete this form (one form per team) and return it to: 
    jordan.cox@oecd-nea.org. 
We must confirm your university country and nationality for your participation in the competition. 

Please complete the form, sign it (either printed or digital), and return it as a PDF file BEFORE beginning any activities on the coding project. 

# Step 2. Inputs and Outputs
In the folder labelled 'inputs' you will find five (5) raw risk registers in excel and pdf format. These are meant to be the input to your code and are paired to the same numbered outputs in the 'outputs' folder. In the outputs folder there are three (3) completed risk registers. 

Your code should ingest the input risk registers and produce the output risk registers using any AI/ML tools that you have access to. These AI/ML tools do not have to be public, but your final code submission must run after importing the appropriate python libraries. All code should be written in Python. 

Your completed code should complete the risk registers for the missing outputs (#4 and #5). 

# Step 3. Submission
In a ZIPPED folder with your team name as the title include only three items:
 - A folder containing your code labelled 'code'
 - Completed risk register 4 labelled '4. Moorgate Crossrail Register (Final).xlsx'
 - Completed risk register 5 labelled '5. Corporate Risk Register (Final).xlsx'
Code must include comments that an expert python coder can understand. The easier your code is to read the better we will be able to verify it runs. 

CODE SUBMITTED WITHOUT NO COMMENTS WILL BE REJECTED FROM THE COMPETITION. 

All submissions are due by midnight CET 27 March 2026 and must be emailed to jordan.cox@oecd-nea.org. 

# Good luck and happy coding!
